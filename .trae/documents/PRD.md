## 1. 产品概述

追更书架是一款面向重度网文读者的移动端追更管理工具，将散落在各平台的追更提醒统一聚合到一个书架界面。解决读者"多平台追更、易漏章、提醒碎片化"的痛点，帮助用户高效管理连载阅读进度。

- 目标用户：日均阅读 2 小时以上、同时在追 5+ 部连载网文的重度读者
- 核心价值：一个书架管所有追更，智能提醒不漏章，安静模式不扰民

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册（本地数据） | 添加/管理书籍、接收提醒、设置安静模式 |

### 2.2 功能模块

1. **追更书架页**：书架列表、添加书籍、状态标签（断更/爆更/待补读）、快速操作
2. **提醒详情页**：新章提醒详情、阅读进度校准操作、更新历史时间线
3. **设置页**：安静模式配置、常用阅读时段、提醒偏好、数据管理

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 追更书架页 | 书架列表 | 展示所有追更书籍卡片，显示书名、作者、来源平台、当前章节、状态标签颜色；支持排序（按更新时间/书名/状态）；下拉刷新检查更新 |
| 追更书架页 | 添加书籍弹窗 | 输入书名、作者、来源平台（起点/番茄/晋江等）、当前读到第几章、手动输入更新时间规律（如"每日22:00"或"每周一三五"） |
| 追更书架页 | 状态标签系统 | 断更（红色标签，超过预期更新时间7天以上）、爆更（金色标签，单日更新3章以上）、待补读（蓝色标签，有未读新章）、正常（绿色标签） |
| 追更书架页 | 底部导航 | 书架/设置两个Tab切换 |
| 提醒详情页 | 新章信息 | 展示第几章、章节标题、字数、距离上次更新多久 |
| 提醒详情页 | 进度校准操作 | 三个按钮："已读到最新"→标记为正常、"稍后再看"→保持待补读、"暂停追更"→暂停该书提醒 |
| 提醒详情页 | 更新历史 | 时间线展示近期更新记录，每条含章节号、标题、字数、更新时间 |
| 设置页 | 安静模式 | 设置上课/会议/睡眠时间段，在此期间提醒合并为一条晚间清单 |
| 设置页 | 阅读时段 | 设置常用阅读时段，系统在此时段优先检查更新 |
| 设置页 | 提醒偏好 | 开关提醒、提醒声音、震动 |
| 设置页 | 数据管理 | 导出/导入书架数据（JSON），清除所有数据 |

## 3. 核心流程

**添加书籍并接收提醒流程**：
用户打开书架 → 点击添加按钮 → 填写书籍信息和更新规律 → 保存到书架 → 系统在阅读时段检查更新 → 发现新章后发送提醒 → 用户点开提醒查看详情 → 选择"已读到最新"/"稍后再看"/"暂停追更" → 书架状态更新

**安静模式流程**：
用户在设置页配置安静时段 → 安静时段内有新章更新 → 不发送单条提醒 → 安静时段结束后推送合并清单（"今日3本小说有更新，共5章待读"）→ 用户点击清单查看详情

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深墨绿 #1A2F23 作为底色，营造沉浸阅读氛围；辅助色：暖米白 #F5F0E8 用于内容区
- 强调色：琥珀橙 #E8913A 用于CTA按钮和重要提示
- 状态色：断更红 #E85D4A、爆更金 #F0B429、待补读蓝 #4A90D9、正常绿 #5CB85C
- 按钮风格：圆角 12px，微投影，触感反馈
- 字体：标题用 Noto Serif SC（宋体气质），正文用 Noto Sans SC，章节号等数字用 JetBrains Mono
- 布局：卡片式，圆角 16px 卡片，卡片间距 12px，顶部吸顶搜索栏
- 图标风格：线性图标，2px 描边，圆角端点

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 追更书架页 | 顶部搜索栏 | 吸顶、深墨绿背景、白色搜索图标、圆角输入框 |
| 追更书架页 | 书架列表 | 竖向滚动卡片流，每张卡片左侧状态色条（4px宽），右侧书名+作者+平台标签+当前章节，底部更新时间和状态标签 |
| 追更书架页 | 添加按钮 | 右下角FAB，琥珀橙圆形按钮，白色加号图标，点击弹出底部Sheet |
| 追更书架页 | 添加书籍Sheet | 从底部滑出，暖米白背景，表单输入项，平台选择用水平滚动Pill按钮，更新规律用快捷模板+自定义 |
| 提醒详情页 | 顶部书籍信息 | 深墨绿背景区，书名大字、作者+平台小字、状态标签 |
| 提醒详情页 | 新章信息卡 | 单张白色卡片，章节号用大号数字、标题居中、字数和更新时长分列底部 |
| 提醒详情页 | 操作按钮组 | 三个等宽按钮横排，分别对应三种操作，颜色区分 |
| 提醒详情页 | 更新时间线 | 竖向时间线，左侧时间节点，右侧章节摘要 |
| 设置页 | 安静时段配置 | 卡片式，每条时段可开关，显示起止时间，支持添加多条 |
| 设置页 | 阅读时段 | 圆形时段选择器，类似闹钟界面 |
| 设置页 | 数据管理 | 列表项样式，带图标，危险操作用红色 |

### 4.3 响应式设计

- 移动优先设计，目标宽度 375px-428px
- 最大内容宽度 480px 居中显示
- 触摸热区最小 44px × 44px
- 支持下拉刷新、左滑删除等移动端手势
